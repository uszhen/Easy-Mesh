# What is Easy-Mesh?
A simple script to set up mesh network based on easytier core

## How to Run

Run the following script on your server (Debian or Ubuntu):
```
bash <(curl -Ls --ipv4 https://github.com/Musixal/easy-mesh/raw/main/easymesh.sh)
```
